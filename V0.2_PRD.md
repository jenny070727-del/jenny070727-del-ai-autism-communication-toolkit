# V0.2 PRD

PRD means Product Requirements Document.

## Goal

Create a safe prototype that uses AI to reduce the effort of making communication support materials based on mature methods.

## Modules

1. Visual Support Generator
2. AAC Communication Card Generator
3. Communication Partner Language Simplifier

## Non-goals

- no diagnosis;
- no treatment claims;
- no behavior-cause explanation;
- no hidden-intention inference;
- no identifiable child data collection.

## Success criteria

- user can understand the project direction in 2 minutes;
- local examples work without API;
- deployed site can call `/api/generate`;
- output remains draft material for adult/professional review.
