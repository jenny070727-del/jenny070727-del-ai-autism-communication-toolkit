# AI 自闭症沟通支持工具包｜一页纸说明 v0.2.2

我目前想做的不是从零发明新的自闭症干预方案，而是基于已有较成熟的沟通支持方法，探索能否用 AI 降低制作和使用门槛。

第一阶段的产品逻辑是：把一个真实沟通场景拆成三件事。

当前交互原则是：用户直接输入真实场景或成人原话，系统生成对应沟通支持材料；预设按钮只作为内部兜底，不作为主入口。

## 1. 视觉流程支持

帮助孩子提前知道“接下来会发生什么”。

参考方法：

- 视觉日程表；
- First-Then / 先后图；
- 社交叙事。

输出：

- 视觉流程；
- First-Then；
- 简短社交叙事；
- 成人展示建议。

## 2. 表达卡系统

帮助孩子表达需要、拒绝、没懂、选择和澄清。

参考方法：

- AAC / 图片沟通；
- 核心词；
- 场景词；
- 功能性沟通。

输出：

- 固定核心卡；
- 当前场景卡；
- 功能表达卡。

## 3. 沟通伙伴支持

帮助成人说得更清楚，并示范如何使用表达卡。

参考方法：

- 沟通伙伴训练；
- 成人语言简化；
- Aided Language Modeling；
- 等待与确认。

输出：

- 低压力短句；
- 成人示范脚本；
- 等待与确认提醒。

## 安全边界

- 不诊断；
- 不治疗；
- 不解释儿童行为原因；
- 不推断隐藏意图；
- 不自动替孩子回复；
- 所有输出都只是草稿，需要成人和专业人员确认。
