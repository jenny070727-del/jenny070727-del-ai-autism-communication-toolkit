# V0.2.2 API Setup

部署后的网站支持：

- DeepSeek
- OpenAI

API 可用于三个模块：

- `visual`：视觉流程支持；
- `aac`：表达卡系统；
- `partner`：沟通伙伴支持。

## 临时浏览器 Key 模式

打开部署后的 Cloudflare Pages 网站，然后：

1. 选择 DeepSeek 或 OpenAI；
2. 粘贴你的 API Key；
3. 点击“保存 Key”；
4. 点击“测试连接”；
5. 在任意模块输入真实场景或成人原话；
6. 点击生成按钮。

Key 只保存在当前浏览器的 localStorage 中，并在请求时发送给 Cloudflare Function。不要公开分享你的 Key。

## Cloudflare 环境变量模式

DeepSeek：

```text
AI_PROVIDER=deepseek
DEEPSEEK_API_KEY=your_key
DEEPSEEK_MODEL=deepseek-chat
```

OpenAI：

```text
AI_PROVIDER=openai
OPENAI_API_KEY=your_key
OPENAI_MODEL=gpt-4.1-mini
```

## 安全提醒

不要在工具里输入可识别儿童身份的信息，例如姓名、学校、联系方式、完整诊断材料、照片或原始聊天记录。
