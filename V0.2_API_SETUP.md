# V0.2 API Setup

The deployed website supports:

- DeepSeek
- OpenAI

## Temporary browser key mode

Open the deployed Cloudflare Pages website, then:

1. choose DeepSeek or OpenAI;
2. paste your API key;
3. click “保存临时 Key”;
4. click “测试连接”;
5. use “用 AI 生成视觉支持材料.”

The key is stored only in the current browser localStorage and sent to the Cloudflare Function for each request. Do not share your key publicly.

## Cloudflare environment variable mode

DeepSeek:

```text
AI_PROVIDER=deepseek
DEEPSEEK_API_KEY=your_key
DEEPSEEK_MODEL=deepseek-chat
```

OpenAI:

```text
AI_PROVIDER=openai
OPENAI_API_KEY=your_key
OPENAI_MODEL=gpt-4.1-mini
```

Do not enter identifiable child information into the tool.
